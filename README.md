## Установка и запуск

### Требования
- Node.js (версия 14 или выше)
- npm
- PostgreSQL (локально или Railway)

### Шаги установки

1. Установите зависимости:
```bash
npm install
```

2. Создайте `.env` на основе примера:
```bash
# PowerShell
Copy-Item .env.example .env
```

3. Настройте `DATABASE_URL` в `.env`.

4. Примените схему БД:
```bash
npm run db:push
```

5. Запустите сервер:
```bash
npm start
```

Или для разработки с автоматической перезагрузкой:
```bash
npm run dev
```

6. Откройте браузер и перейдите по адресу:
```
http://localhost:3000
```

## Структура проекта

- `index.html` - Главная страница
- `styles.css` - Стили сайта
- `script.js` - Клиентская логика
- `server.js` - Backend сервер (Express)
- `prisma/schema.prisma` - Схема PostgreSQL базы данных
- `package.json` - Зависимости проекта
- `railway.json` - Конфиг запуска для Railway

## Деплой на Railway через GitHub

1. Загрузите проект в GitHub-репозиторий.
2. В Railway создайте `New Project` → `Deploy from GitHub repo`.
3. Подключите нужный репозиторий.
4. Добавьте сервис базы данных PostgreSQL в том же проекте Railway.
5. В Variables вашего Node.js-сервиса добавьте:
   - `DATABASE_URL=${{Postgres.DATABASE_URL}}`
6. Убедитесь, что команда запуска:
   - `npm run railway:start`
7. Нажмите Deploy. При старте Railway выполнит `prisma db push`, создаст таблицы и запустит сервер.

После деплоя API и сайт будут работать по публичному URL Railway.
